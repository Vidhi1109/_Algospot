Knapsack

Vidhi Pareek Nov 24,2021

  • Problem

    Given n items, each with a weight and a value, determine maximum value of items that can be obtained by selecting a subset of these objects such that the sum of the weights does not exceed a certain capacity.

  • Solution

    1. Consider all `2^n` possible subsets of items and pick the one whose capacity <= `W` and value is maximum.

    2. Optimal sub-structure (the optimal solution can be constructed efficiently from optimal solutions of its subproblem) : While forming subsets, there are 2 options for each item :

      1. The item is included in the optimal subset.
      2. The item is not included in the optimal subset.
      So at each step , we have 2 choices
      1. Include the current item in the knapsack , increase its value and decrease its capacity. If the capacity goes negative stop else recur.
      2. Exclude the current item from the knapsack and recur for the remaining items.
      3. At every step, we compute both the results and return the maximum value.
      4. Base Case : Either no items are left or the capacity of knapsack became 0. Return 0 in both cases. 
    3. 
//K(w) is the maximum value achievable with capacity w
K(0)=0
for w=1 to W:
  K(w)=max{K(w-wi)+vi : wi <= w}
return K(W)
      4. The above solution will give correct output but due to repetition of similar subproblems time complexity is O(2^n).

  • To achieve a more efficient solution, we should opt for memorisation of the subproblems. We solve smaller subproblems first, then solve larger subproblems from them.
    1. Maintain a 2-D matrix (memorisation table) `T`. Here, `T[i][j]` is the maximum value that can be attained with weight less than or equal to `j` and using items up to first `i` items.
    2. Initialise the `0^(th)` row of the matrix as 0.Consider all rows as items and columns as weight of knapsack from 0 to its capacity.
    3. At each step
      `if` `w[i-1]-j > 0`    `then`    `T[i][j] = max(T[i-1][j], T[i-1][j-w[i-1]] + v[i-1])`
      `else`    `T[i][j] = T[i-1][j]`
    4. Return value T[n][w]
  • Time Complexity: `O(N*W)`` where `N` is the number of weight element and `W` is capacity
