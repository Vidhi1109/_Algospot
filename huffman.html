Huffman Codes

Vidhi Pareek Nov 24,2021


  • Huffman coding is an algorithm used for compressing data. It gives the basic idea of how a file is compressed.It works on greedy paradigm.
  • We know that data is stored in the form of binary strings . The idea is to assign variable length codes to characters.The character which is most frequently used will be assigned the shortest code while the character least used will be assigned longest code.
  • These variable length codes are called prefix codes. These codes are assigned in such a way that the code assigned to one character is not the prefix of code assigned to any other character. This property resolves an ambiguity that might have happened otherwise.
  • For example - Let a , b, c, d be characters and their corresponding codes be 00, 01, 0 and 1. Now if the bitstream is 0001, then the output could be “cccd” or “ccb” or “acd” or “ab”.
  • This algorithm will perform two major tasks:
    1. Build a Huffman tree from input characters
    2. Traverse the Huffman tree and assign codes to characters.

    • PART 1 - BUILDING HUFFMAN TREE


    Input : Characters with their frequencies.
    1. Create a leaf node for each unique character, and build a min heap of all heap nodes.
    2. Extract the two nodes with least frequency from the heap
    3. Now create a new internal node whose frequency is sum of frequencies of these two nodes . This new node should be such that its left child and right child are the previously extracted nodes.
    4. Add this node to min heap. Repeat step 2 and step 3 until the heap contains only one node. The remaining node is the root node and the tree is complete. 
  • 
Create a priority queue Q consisting of each unique character.
sort then in ascending order of their frequencies.
for all the unique characters:
    create a newNode
    extract minimum value from Q and assign it to leftChild of newNode
    extract minimum value from Q and assign it to rightChild of newNode
    calculate the sum of these two minimum values and assign it to the value of newNode
    insert this newNode into the tree
return rootNode

    • PART 2 - Assign codes to characters

    Traverse the tree formed starting from the root. Maintain an auxiliary array. While moving to the left child, write 0 to the array. While moving to the right child, write 1 to the array. Print the array when a leaf node is encountered.

  • Time Complexity Analysis

    O(nlogn) where n is the number of unique characters.
