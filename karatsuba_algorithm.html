Karatsuba Algorithm

Vidhi Pareek Nov 24,2021

The Karatsuba algorithm is a fast multiplication algorithm that uses a divide and conquer approach to multiply two numbers. It happens to be the first algorithm which states that multiplication can be performed at a lower complexity than O(N2). It reduces the complexity of multiplication from O(N2) to O(N1.585). Let us look at how two n-bit integers(say x and y) can be multiplied with complexity O(N1.585).

Explanation

In order to multiply two `n`-bit numbers `x` and `y`. Assume that they have a base `b`,then `x` and `y` can be represented as:

`x=x_1*B^(n/2)+x_0`
`y=y_1*B^(n/2)+y_0`
The product `x*y` becomes

`x*y = (x_1*B^(n/2)+x_0)*(y_1*B^(n/2)+y_0)`
`x*y = x_1y_1B^n+x_1y_0B^(n/2)+x_0y_1B^(n/2)+x_0y_0`

Now we need to find four terms `x_1y_1` , `x_1y_0` , `x_0y_1` , `x_0y_0`.
With careful observation:

`x_1y_0+x_0y_1=(x_1+x_0)(y_1+y_0)-x_1y_1-x_0y_0`

Hence, instead of calculating the four terms stated above , we can calculate `x_1y_1` , `x_0y_0` and `(x_1+x_0)(y_1+y_0)`.
By using this approach, we performed only 3 `n`-bit multiplications instead of 4.

Analysis

For Karatsuba : `T(n)=3*T(n/2)+O(n)`
Time Complexity: `O(N^(log_(2)3))`

By reducing the number of multiplications, this algorithm improvised the time complexity from `O(N^2)` to `O(N^(log_(2)3))`.
