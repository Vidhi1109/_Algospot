Merge Sort

Vidhi Pareek Nov 21,2021

  • Merge sort is one of the most efficient sorting algorithms. It works on the principle of Divide and Conquer.
  • It works on the following principle:
    1. Divide the input array into two halves.
    2. Calls itself for both the halves. In other words, recursively sort both the halves.
    3. Merge the two sorted halves.

    What actually happens?

    • We take variables `i` and `r`to store the starting and ending index of the array.
    • Then we find the middle index of the array using the formula `(i + r)/2`. Let this middle index be stored in variable `q` .Break the array into two subarrays, from `i` to `q` and from `q + 1` to `r` index
    • Divide these arrays recursively just like we did in step 1 and 2.Finally single-element sub-arrays will be obtained.
    • Merge these sorted sub-arrays recursively to obtain a final array.

Time-Complexity Analysis

  • Whenever we divide a number into half in every step, it can be represented using a logarithmic function, which is `log n` and the number of steps can be represented by `log n + 1`(at most).
  • Finding the middle element of array will be a single step operation . So, O(1).
  • To merge the subarrays, made by dividing the original array of n elements, a running time of O(n) will be required.
  • So, the total time for merge sort will become `n(logn + 1)`. Hence, the time complexity becomes `O(nlogn)`.
  • Merge sort will always have a time complexity of `O(nlogn)` i.e its worst case time complexity, best case time complexity and average case time complexity is same.

mergeSort(arr, size)

If  size > 1

Step 1: Find the size of the leftSubArray and rightSubArray so that we can divide the array into two-part
leftSize = size / 2;
rightSize = size - leftSize;
Step 2: Call the mergesort for the leftSubArray
mergeSort(leftSubArray, leftSize);
Step 3: Call the mergesort for the rightSubArray
mergeSort(rightSubArray, rightSize);
Step 4: Call the merge function to merge these two halves
mergeTwoSortedArray(leftSubArray, rightSubArray, arr,leftSize, rightSize)
