Merge Sort

Vidhi Pareek Nov 24,2021

  • Problem

    Given a set S and a collection of sets `S_1 , S_2 ,..., S_n`. All `S_is` are subsets of B. The task is to find the `S_is` such that their union is B. The number of sets picked should be minimised.
  • Greedy Approach:Pick the set `S_i` with maximum number of uncovered elements.Keep doing this until all elements in S are covered.
  • Let's look at an example :
    Let `S={1,2,3,4,5,6}`
    `S_1={1,2,3,4} , S_2={1,3} , S_3={2,4,6}`
    As per our approach, we should pick `S_1` followed by `S_3` follwed by `S_2`.
    Solution : `S_1 , S_2 , S_3`
    But , if we modify `S_2` to `{1,3,5}`, then also the algorithm will pick `S_1` followed by `S_2` followed by `S_3`.
    The solution obtaines in this case is not optimal, since a solution with lesser number of sets could be `S_2` and `S_3`.
  • Set-Cover is NP-Complete.
