Shortest Path in DAG

Vidhi Pareek Nov 24,2021




  • Problem

    Given a weighted,directed,acyclic graph and a source vertex, find the shortest path from source vertex to every other vertex given in the graph

    Note : A Directed Acyclic Graph or a DAG is a directed graph with no cycles.

  • We will use Topological sort to solve this problem.

    What is Topological sort?

    • This works only for DAG.(Directed Acyclic Graph). This sort is an ordering of vertices `v_1,v_2,v_3,....,v_n` of a graph in such a way that if there is an edge from `v_i` to `v_j`, then `v_i` should always come before `v_j` in the topological order of a graph.
    • In other words, the vertices are ordered in ascending order of their indegrees.

  • How to use topological sort to find shortest path?

    Traverse through the vertices in order of their topological sort and for all neighboring vertices of each vertex check
    1. Calculate the length of path when the neighboring vertex is reached through the current vertex.
    2. Compare it with the current value of path length of that vertex.
    3. If the newly calculated value is smaller, update the value of path length. Otherwise , do nothing.
  • Time-Complexity :
    • Topological sorting : `O(v+e)`      where `v` and `e` are number of vertices and edges respectively.
    • While traversing through the sort list : `O(v+e)`      where `v` and `e` are number of vertices and edges respectively.
    • Overall time complexity : `O(v+e)`      where `v` and `e` are number of vertices and edges respectively.
